# 🚀 Crypto Portfolio

Небольшое React-приложение для отслеживания криптовалютного портфеля.

![Preview](https://raw.githubusercontent.com/ogblame/crypto-app/main/frontend/public/assets/screenshots/preview.png)

## 📌 Что реализовано

- 📊 Просмотр списка криптовалют
- 💰 Подсчет общей стоимости портфеля
- 📈 График портфеля
- 📋 Таблица активов
- ➕ Добавление активов
- ⚙️ Управление состоянием через React Context
- 🎨 Интерфейс на Ant Design

## 🛠️ Стек

- React
- JavaScript
- Context API
- Ant Design
- Vite

## 📚 Что изучил

- Архитектуру React-приложений
- Разделение приложения на компоненты
- Работу с Context API
- Передачу данных между компонентами
- Обработку и объединение данных
- Построение интерфейсов с Ant Design

## 🎯 Цель проекта

Проект создан в учебных целях для практики React и построения архитектуры приложения.
